User Manual



Installation


Step 1

We use npm as our 'makefile'. In terminal type 'npm rebuild oracledb'. If you are on a lab machine, there will be errors, this is normal.

Step 2

In terminal type 'node server.js' to start the server.

Step 3

In a browser, go to 'localhost:8080' and login or register



Login and Register

If you have an existing login, then enter your username and password. If you're new, then click on 'Register' and fill in the form details.


We will need the following details: username, password, first name, last name, address, email, and phone. Your username and email must be unique.


After registering, login using your newly created account.



Uploading an Image

To upload an image, navigate to the 'Upload' button on the homepage.

Choose a single or multiple file to be uploaded. Then select permissions, permission 1 will be public, 2 is private, or you can use your group's ID to set to 'group only' view. Subject, location, date taken, and description must also be filled out.



View Images

To view images, navigate to 'Display' on the homepage.

To see more details of an image, click its thumbnail.


There are 3 categories of images:


My Pictures: These are the photos you've uploaded

Group Pictures: These are the photos your group has shared

Popular Pictures: These are the application's most popular photos. Popularity is determined by the number of distinct users who have viewed that image's details


If you are an admin user, there will be a fourth gallery called 'Admin Pictures'. This gallery will contain every photo in the application (even those set to private).



Edit a Photo

Users may only edit their own photo.

To edit, navigate to 'Display' and click on any photo in the 'My Pictures' gallery.

An 'edit photo' button will appear that will take you to the edit page.


In the edit page, you will see the preview of your photo as well as the previous details. To update a photo type in the new values for each field and click on 'EDIT'.


To delete that image, click the red 'DELETE' button at the top of the page



Groups

To view or manage groups, navigate to the 'Groups' on the homepage.

Here you can view the groups you've created (Owned Groups) as well as the groups you're apart of (Memberships). Both tables will list the group's name as well as its unique group ID.


To add a new group, enter a group name in the text box labelled 'Group Name' and select 'Add a group'. Your new group will appear under the Owned Groups table.


To manage a group you own, click on the row under the Owned Groups table.

On this page you can add or delete a member by entering a User's username.

The 'Members of [group name] group' table lists all the members of this group as well as their join date.


To delete a group, select the red 'DELETE' button at the top of the page.


Search for an Image

There are 3 different search scenarios a User can search for:


Scenario 1 : keyword(s) only

To search by keyword(s) only, type in a query in the 'Keywords' text box with different keywords separated by commas. Leave start date and end date blank then click search.

Scenario 2 : time frame only

To search by time frame, enter a start date and end date using the drop down calendar. Leave the keywords text box blank then click search

Scenario 3 : keyword(s) and time frame

To search by both keyword and time frame, simply fill out both the text box and dates then click search



Results for search queries will appear as thumbnails at the bottom of the screen. They can be clicked on to get more details.


Data Analysis (admin only)

This module will be used by the system administrator (a user named 'admin') to generate and display an OLAP report for data analysis. A user of this module may choose to display the number of images for each user, subject, and/or period of time.


To use, enter the fields in which you want an analysis of then click 'GENERATE'.

A table will appear with the resulting data.



Logout

To logout of the application, click the 'LOGOUT' button on the homepage.
